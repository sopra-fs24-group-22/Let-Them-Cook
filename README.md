# Let Them Cook

## Quick Start

### Client

Change to the client directory

```
cd client
```

Install the dependencies

```
npm install
```

Start the client dev server

```
npm start
```

Test the client

```
npm run test
```

Build the client

```
npm run build
```

### Server

Open the project in your IDE and run the com.letthemcook application in debug mode.
